# Activity Tracker 🏃‍♂️

Учебный проект по разработке мобильных приложений. Приложение предназначено для отслеживания физической активности, хранения истории тренировок и анализа статистики.

## 🛠 Стек технологий
* **Язык**: Kotlin
* **Архитектура**: MVVM (Model-View-ViewModel)
* **UI**: Jetpack Compose
* **База данных**: Room (Локальное хранение)
* **Навигация**: Navigation Component
* **CI/CD**: GitHub Actions (Автоматизированное тестирование)

## 📱 Функционал
- Добавление тренировок с указанием типа активности и длительности.
- Отображение истории тренировок.
- Расчет статистики (общая длительность, время по типам активности).
- Навигация между главным экраном и экраном статистики.

## 🧪 Тестирование и CI/CD
Проект включает в себя:
1. **Unit-тесты**: Проверка бизнес-логики в ViewModel.
2. **UI-тесты**: Проверка корректности отображения интерфейса (используются Android Instrumentation Tests).
3. **Автоматизация**: При каждом коммите в репозиторий запускается процесс сборки и тестирования через GitHub Actions.

## 📋 Как запустить
1. Клонируйте репозиторий: `git clone [ссылка]`

3. Синхронизируйте Gradle.
4. Запустите на эмуляторе (API 29+).
